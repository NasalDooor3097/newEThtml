IFRAME Permite incrustar contenido de otra página web dentro de la tuya. src: Especifica la URL del documento que se mostrará en el iframe.

ismap: Indica que la imagen es parte de un mapa de imágenes que utiliza el protocolo HTTP para enviar las coordenadas del clic del usuario al servidor.

srcset: Define una lista de recursos de imágenes con diferentes tamaños para adaptarse a distintas resoluciones y tamaños de pantalla. 		<img src="small.jpg" srcset="large.jpg 1024w, medium.jpg 640w" sizes="(max-width: 600px) 480px, 800px" alt="Descripción de la imagen">

INPUT Permite a los usuarios introducir datos en una página web. La etiqueta <input> puede tener varios tipos y atributos, y su funcionalidad varía según el valor del atributo type.

type: Define el tipo de campo de entrada. Los valores más comunes incluyen:

text: Campo de texto de una sola línea.

password: Campo para contraseñas, oculta el texto ingresado.

email: Campo para direcciones de correo electrónico, con validación de formato.

number: Campo para ingresar números.

date: Campo para seleccionar fechas.

checkbox: Casilla de verificación.

radio: Botón de opción en un grupo.

file: Campo para seleccionar archivos.

submit: Botón para enviar el formulario.

reset: Botón para restablecer los campos del formulario.

button: Botón genérico.

range: Campo para seleccionar un valor dentro de un rango.

tel: Campo para números de teléfono.

url: Campo para URLs.

placeholder: Muestra un texto de sugerencia dentro del campo de entrada cuando está vacío, que desaparece cuando el usuario comienza a escribir.

required: Indica que el campo es obligatorio y debe ser completado antes de enviar el formulario. Este atributo se usa como un booleano.

readonly: Hace que el campo de entrada sea de solo lectura, impidiendo que el usuario modifique su valor. Este atributo se usa como un booleano.

LABEL Se utiliza para definir una etiqueta para un elemento de formulario, como un campo de entrada (<input>), una caja de selección (<select>), o un área de texto (<textarea>).

for: Este atributo se usa para asociar el <label> con un campo de formulario específico mediante el valor del atributo id del campo.
TEXTAREA Se utiliza para crear un área de texto multilínea en un formulario donde los usuarios pueden ingresar texto más largo que no se ajustaría a un campo de entrada de una sola línea.
rows: Define el número de filas visibles en el área de texto.

cols: Define el número de columnas visibles en el área de texto.

placeholder: Muestra un texto de sugerencia dentro del área de texto cuando está vacío.

maxlength: Define el número máximo de caracteres que el usuario puede ingresar en el área de texto.

readonly: Hace que el área de texto sea de solo lectura, impidiendo que el usuario modifique su contenido.

BUTTON Se utiliza para crear un botón interactivo en una página web. type: Define el tipo de botón. Los valores posibles son:

submit: Envía el formulario en el que está incluido. Este es el valor predeterminado si no se especifica ningún tipo.

reset: Restablece los campos del formulario a sus valores iniciales.

button: Un botón genérico que no tiene ninguna funcionalidad predeterminada y puede ser utilizado para ejecutar scripts o realizar otras acciones programáticas.

SELECT Se utiliza para crear un menú desplegable, que permite a los usuarios seleccionar una opción de una lista predefinida. multiple: Permite seleccionar múltiples opciones del menú desplegable. Cuando se usa este atributo, el menú desplegable se convierte en una lista múltiple.

OPTION Se utiliza dentro de un elemento <select> o <optgroup> para definir las opciones que el usuario puede seleccionar en un menú desplegable.
value: Define el valor que se enviará al servidor cuando el formulario se envíe con esta opción seleccionada. Si no se especifica un atributo value, el contenido del texto de la opción se enviará como valor.

selected: Indica que la opción debe estar seleccionada por defecto cuando se carga el formulario. Este atributo se usa como un booleano. Solo una opción puede tener este atributo en un menú desplegable de un solo select.

disabled: Desactiva la opción, haciéndola no seleccionable por el usuario. Esto es útil para mostrar opciones que están deshabilitadas temporalmente.

LEGEND Se utiliza para proporcionar una descripción o título para un grupo de elementos de formulario dentro de un contenedor
Sin atributos

DATALIST Se utiliza para proporcionar una lista de opciones predefinidas para un campo de entrada, permitiendo que los usuarios elijan entre una lista de valores sugeridos.

value: Define el valor que se sugiere para la opción.

label: Proporciona una etiqueta alternativa para la opción, que se muestra en lugar del texto del contenido de la opción.

disabled: Desactiva la opción, haciéndola no seleccionable. Esto puede ser útil para mostrar opciones que no están disponibles actualmente.
OPTGROUP Se utiliza para agrupar un conjunto de opciones dentro de un elemento label: Este atributo especifica el texto que se mostrará como encabezado del grupo de opciones en el menú desplegable. El valor de label actúa como una etiqueta para el grupo de opciones que sigue.

target: Especifica dónde se debe abrir el enlace.

rel: Define la relación entre el documento actual y el documento enlazado.

download: Indica que el enlace debe tratarse como una descarga, en lugar de seguir la URL.

hreflang: Especifica el idioma del recurso al que se enlaza.

type: Indica el tipo de contenido del recurso enlazado.

<a href="https://www.example.com" target="_blank">Visita Example.com</a>

MAP Se utiliza para definir un mapa de imagen interactivo, donde se pueden crear áreas clicables en una imagen que dirigen a diferentes recursos o secciones de una página web. Sinn atributos <img src="imagen.jpg" usemap="#mapa">

AREA La etiqueta <area> en HTML se utiliza dentro de un elemento <map> para definir áreas específicas dentro de una imagen que son interactivas. shape: Define la forma del área clicable. Los valores posibles son:

rect (rectángulo): Define una región rectangular.

circle (círculo): Define una región circular.

poly (polígono): Define una región poligonal con múltiples lados.

Valor: rect, circle, poly.

coords: Especifica las coordenadas para la forma del área. La interpretación de las coordenadas depende del valor del atributo shape:

Para rect: Cuatro valores que representan las coordenadas del punto superior izquierdo y el punto inferior derecho (x1, y1, x2, y2).

Para circle: Tres valores que representan las coordenadas del centro del círculo y el radio (x, y, radio).

Para poly: Un conjunto de coordenadas que definen los vértices del polígono.

Valor: Una serie de números separados por comas.

href: La URL a la que se debe dirigir el usuario al hacer clic en el área. Puede ser una URL externa, una URL interna, o un identificador de anclaje en la misma página.

Valor: Una URL o identificador de anclaje.

alt: Proporciona un texto alternativo para el área, útil para la accesibilidad y para los casos en los que la imagen no se puede cargar. Este texto es utilizado por lectores de pantalla y se muestra si la imagen está ausente.

Valor: Texto descriptivo.

target: Especifica dónde se debe abrir el enlace. Los valores comunes son _blank (para abrir en una nueva pestaña o ventana), _self (para abrir en la misma ventana o pestaña), _parent (para abrir en el marco padre), y _top (para abrir en el marco completo).

Valor: _blank, _self, _parent, _top.

 <map name="mapa">

<area shape="rect" coords="34,44,270,350" href="link1.html">

</map>

PROGRESS se utiliza para mostrar una barra de progreso que indica el avance de una tarea o proceso en curso. value: Define el valor actual del progreso. Este atributo especifica cuánto se ha completado de la tarea en comparación con el valor máximo.

Valor: Un número que representa el progreso actual.

max: Define el valor máximo posible de progreso. Esto establece el total de la tarea que debe completarse.

Valor: Un número que representa el valor máximo.

<progress value="70" max="100">70%</progress>
EMBED Se utiliza para incrustar contenido externo en una página web, como multimedia o aplicaciones interactivas. src: Especifica la URL del recurso que se desea incrustar. Este puede ser un archivo de audio, video, imagen, o incluso un documento.

type: Define el tipo MIME del recurso incrustado, lo cual ayuda al navegador a identificar y manejar el contenido de manera adecuada.

<embed src="video.mp4" width="400" height="300">
METER Se utiliza para representar una medida escalar dentro de un rango conocido, como una cantidad que se ha completado, un nivel de capacidad, o un valor en relación con un rango definido. value: Define el valor actual de la medición. Este atributo especifica el valor que se está midiendo y se representa dentro del rango establecido.

min: Define el valor mínimo del rango de la medición. Este atributo establece el límite inferior del rango en el que se mide el valor.

max: Define el valor máximo del rango de la medición. Este atributo establece el límite superior del rango en el que se mide el valor.

low: Define el valor por debajo del cual la medición se considera baja. Se utiliza para establecer un umbral bajo dentro del rango de medición 		<meter value="0.6" min="0" max="1">60%</meter>
OBJECT Se utiliza para incrustar contenido multimedia o documentos externos dentro de una página web. data: Define la URL del recurso externo que se va a incrustar.

type: Especifica el tipo MIME del recurso incrustado.

form: Asocia el objeto a un formulario específico. 		<object data="archivo.pdf" type="application/pdf" width="600" height="400"></object>
