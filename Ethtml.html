Metadatos del documento
Etiqueta Descripcion Atributos Ejemplos
HEAD Se utiliza para contener metadatos y enlaces a recursos externos que son necesarios para el documento web, pero que no se muestran directamente en la página. sin atributos <head>
|
|
<head>
META Se utiliza para incluir metadatos en el documento web. Estos metadatos proporcionan información sobre el contenido de la página y no se muestran directamente al usuario. Name: Especifica el tipo de metadatos.
Charset: Define la codificación de caracteres del documento.
Content: Proporciona el valor de los metadatos especificados en el atributo name
Http-equiv: Proporciona información equivalente a los encabezados HTTP
Scheme: Especifica un esquema que se usará para interpretar el valor de la propiedad.
Description: Proporciona una breve descripción de la página.
Keywords: Lista de palabras clave relacionadas con el contenido de la página.
Author: Nombre del autor de la página.
Viewport: Configura cómo se debe mostrar la página en diferentes dispositivos. 		<meta charset="UTF-8">
<meta name="description" content="Esta es una página de ejemplo.">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta http-equiv="refresh" content="30"> <meta name="example" content="value" scheme="example-scheme">
LINK Se utiliza para definir la relación entre el documento actual y un recurso
externo. Es especialmente útil para enlazar hojas de estilo CSS, pero también tiene otros usos. 		Rel: Define la relación entre el documento actual y el recurso enlazado.
Stylesheet: Enlaza una hoja de estilo CSS.
Icon: Especifica un icono para la página (favicon).
Alternate: Proporciona una versión alternativa del documento.
Href: Especifica la URL del recurso enlazado.
Type: Define el tipo de contenido del recurso enlazado. Por ejemplo, text/css para hojas de estilo.
Media: Especifica el tipo de dispositivo para el que está diseñado el recurso enlazado
Sizes: Utilizado con rel="icon" para definir las dimensiones del icono.
<link rel="stylesheet" href="styles.css" type="text/css" media="screen">
<link rel="icon" href="favicon.ico" type="image/x-icon">
<link rel="preconnect" href="https://example.com">
<link rel="stylesheet" href="print.css" type="text/css" media="print">
<link rel="stylesheet" href="styles-fr.css" hreflang="fr">
TITLE Define el título de la página web que se muestra en la barra de título del navegador o en la pestaña de la ventana. Este título también suele ser utilizado por los motores de búsqueda como el texto clicable en los resultados de búsqueda. Sin atributos <title>Mi página web</title>
Secciones
Etiqueta Descripcion Atributos Ejemplos
HEADER Define una sección de encabezado en una página o una sección de una página web. Su propósito principal es proporcionar una estructura clara y significativa al contenido de una página, facilitando la organización y la accesibilidad. Aria-label: Proporciona una descripción textual del artículo para tecnologías asistivas, como lectores de pantalla, mejorando la accesibilidad.


Aria-labelledby: Referencia el id de otro elemento que proporciona un encabezado o una descripción para el artículo. Esto puede ser útil si el artículo tiene un título específico en otro elemento.



Aria-describedby: Referencia el id de un elemento que proporciona información adicional sobre el artículo. 		<header>
<h2>Título del Artículo</h2>
<p>Publicado el 10 de septiembre de 2024</p>
</header>
NAV Se utiliza para identificar una sección de enlaces que forman parte de la navegación principal del sitio web. <nav>
<ul>
<li><a href="#home">Inicio</a></li>
<li><a href="#about">Sobre Nosotros</a></li>
<li><a href="#services">Servicios</a></li>
<li><a href="#contact">Contacto</a></li>
</ul>
</nav>
SECTION Se utiliza para definir secciones temáticas dentro de un documento. Es un elemento de bloque que agrupa contenido relacionado en una página web, y cada sección puede tener su propio encabezado, texto, imágenes, etc. <section>
<h2>Introducción</h2>
<p>Bienvenido a nuestra página web.</p>
</section>
ARTICLE Se utiliza para definir contenido independiente y autónomo dentro de un documento. Esta etiqueta es particularmente útil para marcar bloques de contenido que podrían ser distribuidos o reutilizados en otros contextos <article aria-label="Artículo sobre avances en tecnología">
<article aria-labelledby="article-title">
<article aria-describedby="article-summary">

<article>
<p>Este es un breve artículo que proporciona información concisa sobre un tema específico.</p>
</article>
ASIDE Se emplea para incluir información que es relevante para el tema principal, pero que podría ser considerada como un "extra". <aside>
<h2>Nota Adicional</h2>
<p>Este artículo se basa en estudios</p>
</aside>
FOOTER Generalmente contiene información al final de un documento o una sección específica, como la información de contacto. <footer>
<p>&copy; 2024 Mi Sitio Web. Todos los derechos reservados.</p>
</footer>
MAIN contener el contenido único y central del documento. Este contenido es el que describe el propósito principal de la página o la aplicación. <main>
<h2>Bienvenido a nuestro sitio web</h2>
<p>Aquí encontrarás información relevante sobre nuestros productos y servicios.</p>
</main>

Etiqueta Descripcion Atributos Ejemplos
DIV Es un contenedor genérico y versátil que se utiliza para agrupar y organizar otros elementos dentro de un documento HTML. Data-*: Permite almacenar información personalizada en el elemento. Los atributos data-* son útiles para pasar datos a scripts sin afectar el marcado HTML.

Lang: Especifica el idioma del contenido del elemento. Esto puede ser útil para accesibilidad y para la selección de idioma en aplicaciones multilingües.

Dir: Define la dirección del texto en el elemento. Los valores posibles son ltr (de izquierda a derecha) y rtl (de derecha a izquierda).

Title: Proporciona información adicional sobre el elemento. Este texto aparece como un tooltip cuando el usuario pasa el ratón sobre el elemento. 		<div>
<p>Lorem Ipsum es simplemente el texto de relleno de las imprentas y archivos de texto.</p>
</div>
